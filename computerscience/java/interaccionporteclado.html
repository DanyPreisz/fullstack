"Interacción por teclado con el usuario"


Podemos recoger la entrada de teclado y usarla como variable para nuestro programa. La mayor parte de las aplicaciones del mundo real no interactúan con el usuario así, sino por medio de interfaces gráficas, pero escanear la entrada por teclado es una manera muy sencilla de conseguir interacción y nos será muy útil para probar nuestros programas.



Análisis del código


Se crea una nueva clase denominada IntroducciónCaracteresPorTeclado.

Añadimos el método main.

Para leer caracteres introducidos por teclado, usamos System.in.

System.out representa la consola de salida y System.in representa la entrada, en este caso el teclado.

Usamos la clase Scanner que nos ayuda a leer cadenas de caracteres.

Creamos un nuevo Scanner con la línea: Scanner scanner = newScanner (System.in);.

El operador New nos sirve para crear nuevos objetos. En este caso estamos creando un nuevo objeto de tipo Scanner.

Preguntamos al usuario por su nombre: System.out.println ("¿Cómo te llamas?");.

Escaneamos lo que el usuario introduce: String nombre = scanner.nextLine();. Esta instrucción hará que el programa se quede a la espera de que el usuario introduzca una cadena de caracteres por teclado. Al pulsar la tecla Enter, escaneará la línea introducida y lo asignará a una variable nombre, de tipo String. Una String es una cadena de caracteres que en Java es un objeto.

Ahora que sabemos el nombre del usuario, vamos a usarlo para darle la bienvenida: System.out.println ("hola" + nombre + "¡Bienvenido!");. Hemos usado el método nextLine del objeto Scanner. Esto le indica que estamos esperando una cadena de caracteres.

También podemos indicarle al Scanner otro tipo de dato que esperamos del usuario, por ejemplo, un entero. Pedimos al usuario que introduzca su edad: System.out.println ("Introduce tu edad") y la escaneamos: int edad = scanner.nextInt(). Usamos el método nextInt() para indicarle al escáner que estamos esperando un entero. Estamos escaneando el número introducido, y lo almacenamos en la variable edad de tipo int.

Vamos a devolver una respuesta al usuario con su edad: System.out.println ("Tienes" + edad + "años").
