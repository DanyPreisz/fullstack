Datos Primitivos

Son aquellos tipos de datos básicos que no son objetos. Los datos primitivos se utilizan para almacenar valores simples, como números enteros, números en coma flotante, caracteres y booleanos. Estos datos primitivos se definen por el lenguaje y no por el programador.

Tipos de datos primitivos

Cada tipo de dato primitivo tiene un rango de valores que puede almacenar y una forma de representarlos en memoria. También tienen operaciones y métodos específicos para realizar cálculos y manipular los valores almacenados en ellos.

  • boolean: representa un valor booleano verdadero o falso.
  • byte: un entero de 8 bits que puede representar valores entre -128 y 127.
  • short: un entero de 16 bits que puede representar valores entre -32,768 y 32,767.
  • int: un entero de 32 bits que puede representar valores entre -2,147,483,648 y 2,147,483,647.
  • long: un entero de 64 bits que puede representar valores entre -9,223,372,036,854,775,808 y 9,223,372,036,854,775,807.
  • float: un número en coma flotante de 32 bits que puede representar valores entre 1.4E-45 y 3.4028235E38.
  • double: un número en coma flotante de 64 bits que puede representar valores entre 4.9E-324 y 1.7976931348623157E308.
  • char: un carácter Unicode de 16 bits que puede representar cualquier carácter Unicode.

Objetos y no objetos

Para que un dato primitivo se comporte como un objeto se lo escribe con la primera letra en mayúscula. La diferencia entre byte y Byte es que byte es un tipo de dato primitivo que almacena un valor entero de 8 bits, mientras que Byte es una clase envolvente (wrapper class) que encapsula un valor de tipo byte en un objeto. y proporciona métodos y propiedades para interactuar con el valor encapsulado.

Mientras que los tipos de datos primitivos como byte se utilizan para almacenar valores simples de manera eficiente en la memoria, las clases envolventes como Byte se utilizan para representar objetos que se utilizan con mayor frecuencia en la programación orientada a objetos, como colecciones de objetos o valores nulos.

La clase Byte proporciona métodos y propiedades para interactuar con el valor encapsulado en el objeto, como byteValue() para obtener el valor de tipo byte contenido en el objeto Byte, y parseByte(String s) para convertir una cadena en un valor de tipo byte.
