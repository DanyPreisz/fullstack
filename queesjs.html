¿Qué es Java Script?


Java Script es un lenguaje de programación que permite crear aplicaciones web y móviles en múltiples tipos de dispositivos y navegadores web. Es un lenguaje que sigue la especificación ECMA Script y que es, junto a HTML y CSS, una de las tres tecnologías elementales en el mundo de la web.

Java Script fue creado para dar dinamismo a las páginas web que originalmente eran estáticas. La introducción del "motor V8" de Google ha mejorado la velocidad y funcionamiento de Java Script permitiéndole pasar del Front-End al Back-End a través de NodeJs.

Java Script está en cualquier web. Y, aunque el alcance de este lenguaje llega también a dispositivos móviles y otro tipo de "hardware", la web siempre es el punto clave de partida. Lo que se ve en una página web es la combinación de tres tecnologías: HTML, CSS y Java Script.


  1. HTML: Estructura y significado del contenido.
  2. CSS: Presentación del contenido.
  3. Java Script: Comportamiento del contenido.

La primera capa es HTML, que solo da estructura y significado al contenido. Al ser un lenguaje estrictamente de marcado, carece de ciertas características de vista e interactividad.


La segunda capa en una página web es la de CSS. Existe para cuestiones de vista y, aunque también puede disponer de algunas opciones de interactividad, se enfoca principalmente en cómo se verá el contenido, es decir, en su presentación.


La tercera capa corresponde a Java Script. Ya estructurado y bien presentado el contenido, entra en acción Java Script, que es un lenguaje interpretado que se ejecuta en el navegador e interactua perfectamente con las capas de HTML y CSS manipulando e interactuando con el contenido. Desde el más básico clic de un botón hasta una sofisticada animación o la invocación de datos y servicios remotos. Java Script se ha vuelto la piedra angular en el desarrollo web contemporáneo. La evolución de la tecnología ha llevado a extender su uso más allá de un navegador, en el Escritorio y hacia otros dispositivos, garantizando así su forma de trabajo y ejecución.


Siguiendo el orden de las capas, se estructura primero el contenido con HTML, después se dispone de presentación con las reglas expuestas con CSS y, finalmente, se le da vida con la interactividad de la que dispone Java Script. Este lenguaje ha evolucionado maravillosamente desde que fue creado en diciembre de 1995 por Brendan Eich, agregando y mejorando características y elementos que hacen que sea cada vez más fácil y sencillo programar con Java Script. No hay que confundir Java Script con Java. Ambos son lenguajes de programación, pero tienen entornos y contextos muy distintos; aunque la sintaxis en ambos tenga muchas similitudes. El alcance y poder de Java Script es muy amplio, por lo cual existen muchas plataformas y tecnologías donde este lenguaje es fundamental, como el caso de Node o algunos "frameworks" muy poderosos como Angular, React, o Vue, y algunas tecnologías móviles como Native JavaScript o Ionic, entre otras.


Para ser un desarrollador Front-End se debe aprender Java Script, el lenguaje que es responsable de manejar la interacción del usuario con un sitio o aplicación web. Con la evolución de la tecnología, Java Script puede ser también usado en el Back-End. Y como Full-Stack Developer se usa Java Script la mayor parte del tiempo. En una página web existe el DOM (Document Object Model), que ayudará a poder usar e interactuar con todos los elementos de una página como si estos fueran objetos y no solamente etiquetas de HTML.

Jerarquía de DOM

Java Script interactúa con el DOM para manejar los eventos, cargar, mostrar, crear y eliminar todos estos elementos u objetos. Trabajar con Java Script es para aquellos que aprecian crear y diseñar interacciones con contenidos. Permite establecer una relación más fuerte entre el sitio o aplicación web y el usuario final. Al igual que CSS y HTML, Java Script es un lenguaje que ha madurado y evolucionado mucho. Ha tenido muchas revisiones a lo largo del tiempo. Cada navegador web tiene una propia implementación de Java Script. Los desarrolladores hemos tenido que resolver los issues, o problemas, que genera el hecho de que una cierta característica en un lenguaje funcione de una manera en un navegador y de otra manera en otro navegador pues no todos los fabricantes mantienen actualizados constantemente sus navegadores, aunque esta brecha cada vez es menor. Existen técnicas y buenas prácticas con Java Script, como las que incluyen el uso de algunos super sets de este lenguaje, como puede ser TypeScript, de Microsoft, o Go. Otras técnicas que podemos aprovechar y aprender cuando trabajamos con Java Script es la minificación de código o usar algunos flujos de trabajo definidos en herramientas como NodeJS, WebPack o Gulp. Java Script se ha convertido en una forma de poder desarrollar y ejecutar aplicaciones de una manera más sencilla y rápida.
