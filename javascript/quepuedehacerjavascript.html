¿Qué puede hacer JavaScript?

Aplicaciones de la programación en JavaScript

  • Validación del lado del cliente: Permite validar en el Front-End las entradas del usuario antes de enviarlas al Back-End.
  • Manipulación de páginas HTML: Permite manipular la página HTML sobre la marcha, agregando o eliminando etiquetas HTML y modificando el HTML para cambiar su aspecto en función de diferentes dispositivos y requisitos.

    • JavaScript puede cambiar contenido HTML.


    JavaScript puede cambiar valores de los atributos HTML

    En este caso JavaScript cambia el valor del atributo src (source) de una imágen. También puede ocultar o mostrar la imágen.



  • Notificaciones de usuario: Permite generar ventanas emergentes dinámicas (pop-ups) en las páginas web para mostrar mensajes a los visitantes del sitio.


    JavaScript puede mostrar mensajes al usuario.


  • Carga de datos de Back-End: Proporciona la biblioteca Ajax que ayuda a cargar datos de Back-End mientras se realiza algún otro procesamiento
  • Presentaciones: Permite crear presentaciones que brinden aspecto al sitio web.
  • Aplicaciones de servidor: Node JS es una biblioteca basada en eventos que ayuda a desarrollar aplicaciones de servidor muy sofisticadas, incluidos servidores web. Se basa en el tiempo de ejecución de Javascript en el navegador para crear aplicaciones de red rápidas y escalables.

JavaScript es un lenguaje de programación que usamos para que una página web sea más interactiva, para programar aplicaciones móviles y aplicaciones para PC. Fue creado para programar y darle vida a las páginas web. Permite realizar mapas interactivos, animaciones, aplicaciones que se actualizan constantemente, etc. Es responsable de una revolución completa en las páginas web. Sin JavaScript, cada vez que presionáramos un "Me gusta" o hiciéramos una búsqueda en Google tendríamos que esperar que toda la página vuelva a cargarse nuevamente para ver el resultado de la operación ejecutada.

Internet no sería tan útil ni divertido sin JavaScript. Con el tiempo se ha extendido el uso de JavaScript a otras áreas además del desarrollo del FrontEnd. También se usa como lenguaje de BackEnd para plataformas web con conexión a bases de datos, en aplicaciones de teléfonos, computadoras, drones y aplicaciones de realidad aumentada y de realidad virtual. Es uno de los lenguajes con más demanda laboral y es de los más populares porque puede usarse en cualquier escenario. Permite desarrollar páginas interactivas, plataformas web como redes sociales, apps para IOS y Android, software para Windows, Mac o Linux, etc.
