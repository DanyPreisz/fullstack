Suma de impares en un array usando reduce


El método reduce necesita un array y un valor inicial y con cada ejecución opera con el valor que va acumulando con cada elemento del array. Tras operar con todos los elementos del array obtenemos un único valor como resultado. Se reduce el array original a un único valor.

Puede usarse reduce para obtener un único valor a partir de un array. En este caso sería la suma de todos los números impares existentes en el array.

La función sumaTodosImpares recibe como argumento un array de enteros y devuelve la suma de todos los enteros impares.

sumaTodosImpares([1,5,2,9,6,4]); devuelve 1 + 5 + 9 = 15

Lo que tenemos que escribir nosotros es la función reductora, que es la función encargada de ir actualizando el valor del acumulador con cada elemento del array.
