Mostrar resultados o Outputs

JavaScript puede mostrar información de varias formas:

  1. Escribir en un elemento HTML, usando innerHTML. Se cambia la propiedad innerHTML de un elemento HTML.

  2. Usar document.write() para escribir directamente en un documento HTML existente. El HTML especificado se escribirá en el lugar donde se encuentra el script. Sirve para crear contenido dinámico pero no es una función muy utilizada en el desarrollo web moderno. En su lugar, se utilizan bibliotecas JavaScript como jQuery o frameworks como React y Angular para manejar la creación de contenido dinámico y manipulación del DOM.

    Tiene limitaciones, como que no se puede utilizar después de que se haya cargado la página o en el contenido de un archivo externo que se carga después del documento HTML principal. Si se usa después de cargar un documento HTML, elimina todo el HTML existente:

  3. Escribir en un cuadro de alerta, usando window.alert().

    El objeto window es de alcance global. Las variables, propiedades y métodos por defecto pertenecen al objeto ventana. Esto significa que especificar la palabra clave window es opcional.

  4. Escribir en la consola del navegador, usando console.log().

    Se usa en la depuración de código para mostrar resultados en la consola.

  5. Usar window.print().

    JavaScript no tiene ningún objeto de impresión o métodos de impresión. No se puede acceder a dispositivos de salida desde JavaScript. La única excepción es que se puede llamar al método window.print() en el navegador para imprimir el contenido de la ventana actual.
