¿Qué es JavaScript?


JavaScript es un lenguaje de programación que usamos para que una página web sea más interactiva, para programar aplicaciones móviles y aplicaciones para PC. Se usa para programar las páginas web. Permite realizar mapas interactivos, animaciones, aplicaciones que se actualizan constantemente, etc. Es responsable de una revolución completa en las páginas web. Sin JavaScript, cada vez que presionáramos un "Me gusta" o hiciéramos una búsqueda en Google tendríamos que esperar que toda la página vuelva a cargarse nuevamente para ver el resultado de operación ejecutada. Internet no sería tan útil ni divertido sin JavaScript. Con el tiempo se ha extendido el uso de JavaScript a otras áreas además del desarrollo del FrontEnd. También se usa como lenguaje de BackEnd para plataformas web con conexión a bases de datos, en aplicaciones de teléfonos, computadoras, drones y aplicaciones de realidad aumentada y de realidad virtual. Es uno de los lenguajes con más demanda laboral y es de los más populares porque puede usarse en cualquier escenario. Permite desarrollar páginas interactivas, plataformas web como redes sociales, apps para IOS y Android, software para Windows, Mac o Linux, etc.

Existen dos tipos de lenguajes: interpretados o compilados. El lenguaje interpretado no necesita ser compilado. Es una especie de guión o "script" que un intérprete, en este caso el navegador, va ejecutando línea por línea. En cambio, en un lenguaje compilado, el programa desarrollado debe pasar por un punto intermedio donde se "compila" o traduce desde el lenguaje utilizado para escribirlo a un lenguaje que la máquina pueda ejecutar. Con JavaScript la compilación no es necesaria porque el código es directamente interpretado por el navegador. Otra cosa son los frameworks de programación basados en JavaScript como Angular, React, o Vue que si son necesariamente compilados para que el navegador pueda interpretarlos. JavaScript se creó inicialmente para funcionar en un navegador, para que el usuario pudiera interactuar con las páginas web

JavaScript es el lenguaje de programación que permite crear aplicaciones web y móviles a lo largo de múltiples tipos de dispositivos y navegadores web. Es un lenguaje que sigue la especificación ECMAscript y que es, junto a HTML y CSS, una de las tres tecnologías elementales en el mundo de la web. La mejor forma de entender JavaScript es viéndolo en acción. Se lo encuentra en cualquier web. Y, aunque el alcance de este lenguaje llega también a dispositivos móviles y otro tipo de "hardware", la web siempre es el punto clave de partida. Lo que ves en una página web es la combinación de este tridente de tecnologías: HTML, CSS y JavaScript, interpretados por el navegador web. La primera capa es HTML, la cual solo da estructura y significado al contenido que se muestre. Este, al ser un lenguaje estrictamente de marcado, carece de ciertas características de vista e interactividad. Para cuestiones de vista, existe la segunda capa en una página web, la cual es cubierta por CSS y, aunque también puede disponer de algunas opciones de interactividad, se enfoca principalmente en cómo se verá el contenido, es decir, en su presentación. Ya estructurado y bien presentado el contenido, entonces entra en acción la tercera capa, que es cubierta por JavaScript, un lenguaje interpretado que se ejecuta en el navegador e interactua perfectamente con las capas de HTML y CSS manipulando e interactuando con el contenido. Desde el más básico clic de un botón hasta una sofisticada animación o la invocación de datos y servicios remotos.


HTML determina la estructura, CSS la presentación y JavaScript el comportamiento, lo que hace la página.

JavaScript se ha vuelto la piedra angular en el desarrollo web contemporáneo. La evolución de la tecnología ha llevado a extender su uso más allá de un navegador, en el Escritorio y hacia otros dispositivos, garantizando así su forma de trabajo y ejecución. Siguiendo el orden de las capas que hemos mencionado, se estructura primero el contenido con HTML, después se dispone de presentación con las reglas expuestas con CSS y, finalmente, se le da vida con la interactividad de la que dispone JavaScript. Este lenguaje ha evolucionado maravillosamente desde que fue creado en diciembre de 1995 por Brendan Eich, agregando y mejorando características y elementos que hacen que sea cada vez más fácil y sencillo programar con JavaScript. No hay que confundir JavaScript con Java, que, si bien ambos son lenguajes de programación, tienen entornos y contextos muy distintos; aunque ahora la sintaxis en ambos tenga muchas similitudes. Lo que sí es muy cierto: el alcance y poder de JavaScript es muy amplio, por lo cual existen ya muchas plataformas y tecnologías donde este lenguaje es su base fundamental, como el caso de Node o algunos "frameworks" muy poderosos e importantes como Angular, React, o Vue, y algunas tecnologías móviles como Native JavaScript o Ionic, entre otras.
