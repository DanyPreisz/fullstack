Función sólo números

La funcion llamada 'soloNumeros' recibe como argumento un arreglo de enteros y strings llamado 'array'. Debe devolver un arreglo con solo los enteros.

Por ejemplo: soloNumeros([1, 'Henry', 2]) debe retornar [1, 2]

En la Solución 1, la función "soloNumeros" toma un array como parámetro y devuelve un nuevo array que contiene sólo los elementos del array original que son de tipo "number" (números).

La función comienza creando un nuevo array vacío llamado "arrayResultante" que se utilizará para almacenar los números del array original.

A continuación, utiliza un bucle for para recorrer cada elemento del array original. Dentro del bucle, se comprueba si el tipo de dato del elemento actual es un número utilizando el operador "typeof". Si el elemento es de tipo "number", se agrega al array resultante utilizando el método "push". Finalmente, la función devuelve el array resultante que contiene sólo los elementos del array original que son números.

La función soloNumeros se llama con el arreglo [1, 'Henry', 2]. La función filtra solo los valores numéricos del arreglo y devuelve [1, 2].

En la Solución 2 se utiliza el método filter() para iterar sobre el arreglo y filtrar solo los valores que son de tipo number. Luego, devolver el nuevo arreglo que contiene solo los valores numéricos.

Explicación:

  • La función soloNumeros toma un arreglo como argumento llamado array.
  • El método filter() se utiliza en el arreglo array y se pasa una función de devolución de llamada como argumento.
  • La función de devolución de llamada toma un elemento como argumento y devuelve true si el elemento es un número y false en caso contrario.
  • La función filter() crea un nuevo arreglo que contiene solo los elementos que cumplen la condición en la función de devolución de llamada.
  • Finalmente, la función soloNumeros devuelve el nuevo arreglo que contiene solo los valores numéricos.
