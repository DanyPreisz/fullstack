Variables

Hasta ahora calculamos y mostramos un resultado pero no lo guardamos en ningún lado. Si queremos guardar un resultado para poder trabajar con él después, necesitamos usar variables.

El nombre de una variable debe ser corto y descriptivo. Debe cumplir varias condiciones:

  • No debe contener espacios.
  • Debe incluir solo letras, números y guiones bajos.
  • Debe iniciar con una letra o guión bajo.

Para crear una variable se usa = seguido del valor que deseamos que tenga. Se puede ver el valor asignado usando print() y colocando el nombre de la variable entre los paréntesis.
